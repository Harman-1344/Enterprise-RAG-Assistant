Architecture & Flow

1

Document Ingestion

Parses uploaded PDFs and intelligently chunks the text using RecursiveCharacterTextSplitter to maintain context windows.

2

Deep Vector Embeddings

Converts text chunks into mathematical vectors using HuggingFace's all-MiniLM-L6-v2 embedding model.

3

Storage

Stores embeddings locally in a FAISS index for efficient similarity search.

4

Generative Synthesis

Retrieves the top most relevant document chunks based on the user's query and passes them to the LLM to generate a factual answer.

Tech Stack

Language: Python 3.10
AI Framework: LangChain
Embeddings: HuggingFace (sentence-transformers)
Vector Store: FAISS CPU
LLM Engine: Groq API (llama-3.1-8b-instant)
Frontend UI: Streamlit
